# 🎓 GCUF Admission Form

A simple and responsive **University Admission Form** built using **pure HTML5**. This project was created as part of my Web Development learning journey to practice HTML forms and related elements without using CSS or JavaScript.

---

## 📌 Features

- Student Information Section
- Contact Information Section
- Academic Information Section
- Document Upload Section
- Declaration Checkbox
- Form Validation using HTML5
- Organized using Fieldsets and Legends
